# AI-Augmented Real-Time Detection System for Android Ransomware

## Overview
This project presents an AI-Augmented Real-Time Detection System for Android Ransomware using Machine Learning models like Random Forest and XGBoost.

## Features
- Real-time ransomware detection
- Machine Learning based classification
- Dataset training and evaluation
- Android malware analysis

## Project Structure
- dataset/ → Training dataset
- notebooks/ → ML training and detection code
- models/ → Saved trained models
- config/ → Real-time configuration files
- report/ → Project report

## Technologies Used
- Python
- Machine Learning
- XGBoost
- Random Forest
- Jupyter Notebook
